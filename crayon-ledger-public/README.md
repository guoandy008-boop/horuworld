# 蜡笔小账本

蜡笔小账本是一个从个人需求出发做出来的轻量记账 Web App。它先解决“每天能不能顺手记一笔”，再逐步加入多用户、飞书记账、每日小报、预算和报表。

这个仓库用于展示一个真实小产品从 MVP 到可公开版本的过程。

## 功能

- 邮箱注册 / 登录
- 支出、储蓄、预算记录
- 月度统计和分类占比
- 飞书单聊机器人记账
- 飞书绑定码，避免多用户串账
- 每日小报定时推送
- 数据备份脚本
- 账户管理后台

## 技术栈

- 前端：原生 HTML / CSS / JavaScript
- 后端：PHP
- 数据：JSON 文件存储
- 集成：飞书自建应用、飞书 Webhook

## 本地或服务器运行

1. 准备一个支持 PHP 的运行环境。
2. 复制配置文件：

```bash
cp config.example.php config.php
```

3. 修改 `config.php` 中的 `CRON_TOKEN`、`ADMIN_PASS` 和飞书相关配置。
4. 打开 `index.html` 使用。

更多部署细节见 [DEPLOY.md](DEPLOY.md)。

飞书机器人配置见 [FEISHU_SETUP.md](FEISHU_SETUP.md)。

## 版本记录

## 安全说明

本仓库不包含真实配置。不要提交：

- `config.php`
- 飞书 Webhook
- 飞书 App Secret
- 定时任务 token
- 用户数据和账单数据

公开部署前请先阅读 [DEPLOY.md](DEPLOY.md)。
