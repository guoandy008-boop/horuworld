# 蜡笔小账本 · 飞书记账配置教程

这份文档用于配置飞书自建应用和单聊机器人记账。

本教程分两部分：
- **第一部分（管理员看）**：你来配置一次「蜡笔小账本」机器人，让所有用户都能用。
- **第二部分（发给用户看）**：用户怎么绑定、怎么记账（很简单，一段话）。

> 架构说明：**只用你这一个机器人**，所有用户共用它。每个用户在 app 里拿一个「绑定码」，
> 到飞书**单聊**机器人发「绑定 + 码」，服务器就知道之后他发的账该记到谁名下，互不串。

---

# 第一部分：管理员配置（你做一次）

## 一、创建企业自建应用

1. 电脑打开 **open.feishu.cn** → 登录 →「开发者后台」→「创建企业自建应用」。
2. 填名称（蜡笔小账本）、选图标，创建。
3. 进入后在「**凭证与基础信息**」记下 **App ID** 和 **App Secret**。

【配图：凭证与基础信息，标出 App ID / App Secret】

## 二、开启机器人能力

「添加应用能力」→ 启用「**机器人**」。

【配图：启用机器人】

## 三、权限（关键，单聊必须开）

「权限管理」里开通：
- **获取与发送单聊、群组消息**（`im:message`）
- **以应用身份发消息**（`im:message:send_as_bot`）
- **读取用户发给机器人的单聊消息**（单聊消息权限，**务必开这个**，否则用户单聊机器人收不到）

【配图：权限管理，已开通上述权限】

## 四、事件订阅

「事件与回调」→「事件配置」：
1. 订阅方式：将事件发送至开发者服务器。
2. 请求地址填（这是**你的服务器**，所有用户共用，不要改成别的）：

   ```
   http://你的服务器地址/feishu.php
   ```
   （上线 https 后改成 https://你的域名/feishu.php）

3. 验证通过后，加密策略里 **Encrypt Key 留空**。
4. 「添加事件」里添加：**接收消息 `im.message.receive_v1`**，并在它的「查看其他权限」里
   确认**单聊消息**那项也已勾选。

【配图：事件配置 + 已添加接收消息事件 + 单聊权限已勾选】

## 五、设置可用范围 + 发布

1. 把应用的「**可用范围**」设为需要使用的成员（或全员）——这样他们才能在飞书里找到并单聊这个机器人。
2. 「版本管理与发布」→ **创建版本并发布**，看到「当前修改均已发布」才算生效。
   （以后只要改了权限/事件，都要重新发布一次。）

【配图：可用范围设置 + 已发布状态】

## 六、把 App ID / Secret 填进服务器

打开服务器 `config.php`，填：
```php
define('FEISHU_APP_ID',     '你的 App ID');
define('FEISHU_APP_SECRET', '你的 App Secret');
define('FEISHU_OWNER_EMAIL','你自己注册 app 用的邮箱'); // 你本人不绑定也会记到这个账号
```

# 第二部分：用户怎么用（把下面这段发给用户）

### 三步开始用飞书记账 🐷

1. 打开蜡笔小账本 app →【我的】→【数据】→「**飞书记账**」→ 点「**获取绑定码**」，得到一个 6 位数字。
2. 在飞书里**单聊**「蜡笔小账本」机器人，发送：

   > 绑定 你的6位码

   机器人回「绑定成功」即可。
3. 以后**单聊机器人**直接发就能记账，例如：
   - `午餐35`、`打车23`、`奶茶18`
   - `工资到账8000`、`存5000 工资`、`取2000`
   - 发「帮助」看用法

绑定后，每天早上还会**私聊推送**你的「昨日小报」。

### 怎么找到「蜡笔小账本」机器人？

在飞书顶部**搜索框**输入「**蜡笔小账本**」→ 在结果里选「**机器人 / 应用**」分类 → 点进去就能单聊。
（也可以在飞书左侧或「工作台」里找「机器人 / 应用」入口）

> **搜不到机器人？** 说明你还不在该应用所属的飞书组织里。本功能目前**仅限同一个飞书组织内的成员**使用，
> 请联系管理员把你**邀请进该飞书组织**、并把应用对你开放。
> 如果你用的是别的公司的飞书，暂时无法跨组织使用飞书记账——但你仍可以正常用 app 本身（网页/手机）记账。

## 常见问题

- **发了没反应？** 先确认：管理员是否已开通**单聊消息权限并重新发布**；你是否在**单聊**（不是群里）发的；是否已绑定。
- **机器人提示还没绑定？** 先按上面三步绑定。未绑定时不会入账，避免记到别人的账本里。
- **换账号了？** 在 app 飞书记账里「解绑」，再用新账号重新获取绑定码绑定。

> App Secret 属于敏感信息，不要外发；泄露就去飞书后台重置后再填回 config.php。

## 附：关于「让其他公司的人也能用飞书记账」

当前机器人是「**企业自建应用**」，**只能你自己飞书组织内的成员用**。想让外部任意企业的人也能用，需要升级为「**应用商店应用（ISV）**」，门槛较高：

- 用一个企业注册「商店应用」（个人可免费注册企业，需实名）
- 改造为**多租户 + OAuth 授权**（每个安装的企业一套 token、数据隔离）
- 提交隐私政策、权限说明等，**经飞书官方审核上架**
- 面向公众需备齐隐私协议、用户协议等合规材料

**建议**：现阶段把飞书记账当作"同组织成员"的加值功能；要扩大就把人**邀请进你的飞书组织**。
app 本身（网页/手机记账）才是面向所有人的主力，注册即用、不受组织限制。
